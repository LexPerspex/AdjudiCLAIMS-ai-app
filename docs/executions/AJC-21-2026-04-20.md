## Summary

Phase 2 adds PDF generation to the insurance-claims-case-generator package. All 25
`DocumentType` enum values now have registered reportlab PDF generators in three tiers:
Tier A (form-accurate approximations), Tier B (structured layout), Tier C (plain letterhead).
Adds ZIP export, ThreadPoolExecutor batch builder, and CLI `batch` command.

---

## Architecture: Document Registry

```
documents/
├── __init__.py                — exports DocumentRegistry, register_document
├── registry.py                — @register_document decorator + dispatch
├── loader.py                  — imports all 25 generators (triggers registration)
├── base_document.py           — Abstract DocumentGenerator(ABC), _build_doc()
├── pdf_primitives.py          — Shared flowables: hline, para, label_value_table, section_table
├── letterhead.py              — carrier_header_block, wcab_caption, confidentiality_footer
└── form_renderer.py           — Tier A helpers: form_row, form_section_header, checkbox_field
```

Registration happens at import time. `loader.py` is imported by `case_builder.py` when
`generate_pdfs=True`, which triggers all `@register_document` decorations.
