## UPL Compliance Notes

- `RecentBlocksTable` shows only: timestamp, user name, user ID prefix, query
  length bucket, adversarial flag. Query content is **never** logged, stored,
  or displayed. This is factual metrics data (GREEN zone).
- The alert banner shows quantitative rate/count data only â€” no legal conclusions.
- `AlertConfigPanel` is gated to SUPERVISOR+ via `requireRole` on the backend
  and `isSupervisorOrAdmin` on the frontend.
