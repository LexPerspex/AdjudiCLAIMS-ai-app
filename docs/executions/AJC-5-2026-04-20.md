### Data-Shape Mismatches Fixed

The prior `use-compliance.ts` defined a flat `ComplianceMetrics` interface with fields like `overallScore`, `trend`, `deadlineAdherence` (number), `trainingCompletion`, `uplCompliance`, `monthlyReviewDue` — none of which match the actual API responses.

**Actual API shapes** (from `server/services/compliance-dashboard.service.ts`):

- `GET /compliance/examiner` → `ExaminerComplianceMetrics` with nested `deadlineAdherence: DeadlineAdherence`, `uplSummary: UplSummary`, `activeClaimsCount: number`
- `GET /compliance/team` → `SupervisorTeamMetrics` with `teamDeadlineAdherence`, `teamUplCompliance`, `trainingCompletion`, `examinerBreakdown[]`
- `GET /compliance/admin` → `AdminComplianceReport extends SupervisorTeamMetrics` with `doiAuditReadinessScore`, `complianceScoreBreakdown`
- `GET /compliance/upl` → `UplMonitoringMetrics` with `zoneDistribution`, `blocksPerPeriod[]`, `adversarialDetectionRate`

### New UI Components

**ExaminerComplianceView** (updated):
- Shows deadline adherence (met/missed/pending counts) and UPL interaction summary
- Derives overall score from adherenceRate (50%) + UPL compliance rate (30%) + active claims signal (20%)
- Highlights UPL block count with error styling when > 0

**TeamComplianceView** (updated):
- Uses `SupervisorTeamMetrics` shape
- Score ring derived from adherenceRate, blockRate, trainingCompletionRate
- Examiner breakdown shows per-examiner deadline adherence + UPL block rate

**AdminComplianceView** (new — CLAIMS_ADMIN only):
- DOI audit readiness score banner (0–100) with color coding: green ≥90, amber ≥70, red <70
- Per-category score breakdown with progress bars (deadline 40pts, investigation 30pts, docs 20pts, UPL 10pts)
- Org-wide stat grid (deadline adherence %, training completion, UPL block rate, exam status)
- Mandatory disclaimer: "Metrics computed from system data. Consult qualified counsel for regulatory compliance determinations."

**UplMonitoringPanel** (new — CLAIMS_SUPERVISOR+):
- Zone distribution bars with counts (GREEN/YELLOW/RED)
- Adversarial detection rate with 3-tier risk interpretation (<10% normal, 10-30% elevated, 30%+ policy review required)
- Daily blocks sparkbar chart (bar chart grouped by calendar day)

### UPL Compliance

All views display factual system metrics only (GREEN zone). No legal conclusions or analysis. The DOI audit readiness banner and panel include the mandatory disclaimer pointing users to qualified counsel for compliance determinations.
